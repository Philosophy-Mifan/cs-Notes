# Part I HTML

## Chapter 1 HTML的基本属性

在对html开始学习之前，要知道`.html`以及`.htm`都是HTML文档的后缀名，它们之间没有任何区别。

先来看一段简洁的HTML

```html
<!DOCTYPE html>			<!-- 声明为HTML5文档 -->
<html>
    <head>
        <meta charset="utf-8">
        <title>どんな結末がお望みだい？</title>
    </head>
    <body>
        <h1>first title</h1>
        <p>first pragraph</p>
    </body>
</html>
```

`<html>`元素是HTML页面的根元素；
`<head>` 元素包含了文档的元（meta）数据，如 `<meta charset="utf-8">` 定义网页编码格式为 **utf-8**；
`<title>` 元素描述了文档的标题；
`<body>` 元素包含了可见的页面内容；
`<h1>` 元素定义一个大标题；
`<p>` 元素定义一个段落。
